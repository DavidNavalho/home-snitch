# ASUS RT-AX88U Admin Notes

## Access

Admin URL: http://router.asus.com

Fallback LAN URL: http://192.168.50.1

## Reset Caution

Factory reset can erase wireless names, passwords, WAN settings, DHCP reservations, and port forwards. Back up the configuration and confirm the admin password before pressing reset.
