# S7-1200 Preventive Maintenance

## Daily

- Confirm RUN LED is steady green and SF LED is off.
- Check that the diagnostic buffer has no new red entries.

## Quarterly

- Back up the project from TIA Portal to the maintenance vault. Tag the
  archive with the date and the change ticket number.
- Inspect the CPU memory card for corrosion.
- Verify firmware version matches the production-approved baseline.

## Annually

- Replace the CPU clock backup battery if the maintenance LED has been on.
- Reseat all screw-style I/O terminal connectors after a thermal cycle.
