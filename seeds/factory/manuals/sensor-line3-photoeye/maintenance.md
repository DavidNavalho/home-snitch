# SICK W4-3 Preventive Maintenance

## Daily

- Wipe the lens with a microfiber cloth.
- Confirm the green Stab LED is steady when a target box passes.

## Weekly

- Verify the reflector mount has not drifted (check the reference paint mark).
- Re-teach background suppression if the line has been swapped to a new product.

## Spare Parts

- Sensor body `WL4-3F2230`
- Reflector tape `PL30A`
- M12 5-pin cable `DOL-1205-G02M`
